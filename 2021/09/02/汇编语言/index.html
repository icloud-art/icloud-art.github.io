汇编语言

发表于 2021-09-02
字数统计: 659 | 阅读时长 ≈ 2

编程语言的发展

机器语言

  1. 由0和1组成

汇编语言(Assembly Language)

  1. 用符号代替了0和1，比机器语言便于阅读和记忆

高级语言

  1. C/C++/Java/Swift等，更接近人类自然语言

操作：将寄存器BX的内容送入寄存器AX

  1. 机器语言：1000100111011000
  2. 汇编语言：mov ax, bx
  3. 高级语言：ax = bx

程序是如何运行在计算机上面的

  1. 汇编语言和机器语言是一一对应，每一条机器指令都有与之对应的汇编指令
  2. 汇编语言可以通过编译得到机器语言，机器语言可以通过反编译得到汇编语言
  3. 高级语言可以通过编译得到汇编语言/机器语言，但是汇编语言/机器语言几乎不可能还原成高级语言

汇编语言的特点

  1. 可以直接访问、控制各种硬件设备、比如存储器、CPU等，能最大限度的发挥硬件功能
  2. 汇编指令是机器指令的助记符，和机器指令一一对应。每一种CPU都有自己的指令集/汇编指令集，所以汇编语言不具备可移植性
  3. 知识点太多，开发者需要对CPU等硬件结果有所了解，不易于编写、调试、和维护
  4. 不区分大小写，比如mov和MOV是一样的

汇编语言 VS 高级语言

汇编语言的用途

  1. 编写驱动程序、操作系统
  2. 对性能要求极高的程序或者代码片段、可以和高级语言混合使用（内联汇编）
  3. 软件安全：病毒分析与防治；逆向、加壳、脱壳、破解、外挂、免杀、加密解密、漏洞、黑客
  4. 是理解计整个算机系统的最佳起点和有效途径
  5. 弄清代码的本质：sizeof ; ++a + ++a + ++a; switch 和 if 究竟谁的效率高，为什么？

汇编语言的种类

  1. 8086汇编（8086处理器是16Bit的CPU）
  2. Win32汇编
  3. Win64汇编
  4. AT&T汇编（Mac、 iOS模拟器）
  5. ARM汇编（嵌入式、iOS设备）

汇编语言的组成

汇编语言发展到现在，有以下三种指令组成

  1. 汇编指令：机器码的助记符，有对应的机器码
  2. 伪指令：没有对应的机器码，由编译器执行，计算机并不执行
  3. 其他符号：如+ - * / 等，由编译器识别，没有对应的机器码
    汇编语言的核心是汇编指令，它决定了汇编语言的特性。
